What's inside

Everything you need to ship 3D on the web.

⛰

Procedural Terrain

FBM noise + domain warping generates infinite landscapes with height-based colour blending, fog, and specular water gloss.

💡

Custom Phong Lighting

Hand-written GLSL ambient, diffuse and specular — not Three.js built-ins. Full control over shininess, light colour and position per frame.

✦

Bloom + Blur

Separable Gaussian blur and multi-step bloom with configurable threshold and intensity. Toggle on/off at runtime.

🎥

Free-fly Camera

WASD + mouse look with smooth velocity damping. Pointer Lock API. Switchable to orbit mode. Drop it into any scene in two lines.

⚡

Vite + ES Modules

Instant HMR, tree-shaking, and zero config. GLSL loaded via fetch — no bundler plugin needed. Works with any modern framework.

🧪

Fully Tested

Vitest unit tests for every module. Camera, controls, post-processing passes — all verified without a browser or GPU. CI ready.

Quick start

Add 3D to your site in minutes.

Clone the repo, copy the src folder, and import App into your project. Point it at a canvas element and the terrain renders immediately. No WebGL boilerplate. No shader compilation pipeline to configure.

Read the docs → 
// 1. Import the App
import { App } from './src/App.js';

// 2. Mount to any canvas element
const canvas = document.querySelector('#c');
const app = new App(canvas);

// 3. Tune bloom live — no reload needed
app.post.bloomPass.threshold = 0.5;
app.post.bloomPass.intensity  = 1.8;

// 4. Swap in your own GLSL shader
const mat = await PhongMaterial.create({
  shininess: 128,
  lightPos: new THREE.Vector3(50, 80, 50),
});
yourMesh.material = mat.getMaterial();
